1) User registration -
  route - '/signup'
  type - POST
  body parameters - {
                    "name": "name of the user",
                    "email": "email of the user",
                    password: "password",
                    "dob": date of birth in (yyyy-mm-dd) format,
                    "description":"description"
                    }

2) User Login -
  route - '/login'
  type - POST
  body parameters - {
                    "email": "email of the registered user",
                    "password": "password"
                    }


3) Update Profile - (protected route)
  route - '/updateUser'
  type - PUT
  headers -{
      token: value
  }
  body parameters - {
    "name":"name",
    "dob":"dob",
    "description":"description"
}


4) User Profile - (protected route)
  route - '/profile'
